# Mini Channel Talk — KPI 시트

## 1. 7개 지표

★ = 미니 채널톡이 보여주려는 두 핵심(컨텍스트 누적, 자기 해결)과 직접 닿는 지표.

| # | 지표 | 정의 | 무엇을 의미하는가 |
|---|---|---|---|
| 1 ★ | **재인식률** | 두 번 이상 방문한 고객의 비율 | 컨텍스트가 실제로 쌓이는지 직접 보여줍니다. 낮으면 식별 메커니즘이 약합니다. |
| 2 ★ | **프로필 채워짐 비율** | 이름·이메일이 채워진 고객의 비율 | 익명에서 실명으로 넘어간 비율. SMB가 마케팅·재방문 관리를 시작할 수 있는 출발점. |
| 3 ★ | **셀프 해결률** | 워크플로우 끝에서 “도움이 됐어요”로 끝난 비율 | 운영자 호출 없이 고객이 스스로 해결한 비율. 낮으면 분기 설계나 안내 문구가 약합니다. |
| 4 | **TTFR (median)** | 첫 고객 메시지 → 첫 운영자 응답까지 시간의 중앙값 | 응답성. 고객 만족도의 선행 지표. |
| 5 | **활성 대화 수** | 지금 응대중인 대화 수 | 운영 부하. 임계 초과 시 인력 부족 신호. |
| 6 | **재오픈률** | 종료된 대화가 다시 열린 비율 | “한 번에 해결 못 함”의 후행 지표. 높으면 응대 비용 증가. |
| 7 | **운영자 1인당 동시 응대 수** | 운영자 한 명이 동시에 들고 있는 대화의 평균 | 너무 적으면 비효율, 너무 많으면 과부하. |

## 2. 대시보드 6장

데모에서 *눈에 보이게 변하는* 6개를 운영자 콘솔 상단에 두었습니다.

| 카드 | 변하는 시점 |
|---|---|
| 재인식률 ★ | 시나리오 I — 브라우저 닫고 재방문 |
| 프로필 채워짐 ★ | 시나리오 C — 이름·이메일 입력 |
| 셀프 해결률 ★ | 시나리오 H — 워크플로우 끝에서 “도움이 됐어요” |
| TTFR (median) | 운영자 응답마다 자동 갱신 |
| 활성 대화 | 실시간 |
| 재오픈률 | 시드 데이터 기준값 + 데모 중 변화 |

7번 “운영자 1인당 동시 응대 수”는 데모가 단일 운영자 환경이라 시연 가치가 약해 시트에만 두었습니다. 다중 운영자 환경에서 가장 먼저 추가될 카드입니다.

각 카드의 **i** 표시를 누르면 정의·측정법·고객사 의미가 안내창으로 열립니다. 대시보드는 30초 주기로 갱신합니다 — 활성 대화만 실시간에 가까운 가치가 있고, 나머지는 천천히 변하는 지표라 30초로 충분했습니다.

## 3. 측정방식 SQL — 핵심 3개

나머지 4개의 전체 쿼리는 `lib/kpi.ts` 참조.

### 3.1 재인식률

```sql
SELECT
  COUNT(*) FILTER (WHERE visit_count > 1)::float
    / NULLIF(COUNT(*), 0) AS recognition_rate
FROM customers;
```

### 3.2 프로필 채워짐 비율

```sql
SELECT
  COUNT(*) FILTER (WHERE identified_at IS NOT NULL)::float
    / NULLIF(COUNT(*), 0) AS profile_filled_rate
FROM customers;
```

### 3.3 셀프 해결률

```sql
SELECT
  resolved::float / NULLIF(resolved + escalated, 0) AS deflection_rate
FROM (
  SELECT
    COUNT(*) FILTER (WHERE event_type = 'resolved')  AS resolved,
    COUNT(*) FILTER (WHERE event_type = 'escalated') AS escalated
  FROM workflow_events
) c;
```

## 4. 제외한 KPI 후보

| 후보 | 제외 사유 |
|---|---|
| 활성화 (위젯 설치 후 첫 메시지까지) | 데모 환경에서 측정 어려움 |
| CSAT (만족도) | 미니 데이터 양에서 노이즈 큼 |
| 평균 대화 turn 수 | 핵심 가설 검증과 직접성 약함 |
| 시스템 에러율·지연 시간 | FDE 시각이 아닌 SRE 시각 |
